AMD µProf
=========

The AMD µProf offers Command Line Interface to perform CPU and Power profiling functionalities on AMD’s CPUs, APUs and discrete GPUs. On Intel processors, limited CPU Profiling capabilities are supported.

It is supported on the following OS:

    * Windows 7 (64-bit)
    * Windows 8.1 (64-bit)
    * Windows 10 (64-bit)
    * Windows 11 (64-bit)
    * Windows Server 2019 (64-bit)
    * Windows Server 2022 (64-bit)
    * Ubuntu 15.04 & later (64-bit)
    * RHEL 7.0 & later (64-bit)
    * FreeBSD 12.2 & later (64-bit)


How to Run?
-----------
On Windows, the GUI tool AMDuProf.exe and the CLI tool AMDuProfCLI.exe are available at “C:\Program Files\AMD\AMDuProf\bin”.

On Linux, the Command Line Tools are installed at /opt/AMDuProf_<version-number>/bin.


Updates and support
-------------------
The latest version and updates for AMD µProf can be found at: https://developer.amd.com/amd-uprof/
